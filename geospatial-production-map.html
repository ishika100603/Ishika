One step before this loads

This page needs your own Mapbox access token — the same one your assignment brief asks you to set up (step 3, using your own Mapbox access token).

Get a free token at account.mapbox.com/access-tokens, then paste it into the script near the top of this file:

const MAPBOX_TOKEN = 'YOUR_MAPBOX_ACCESS_TOKEN_HERE';

Save the file and reopen it — the map and the delivery network will render.
